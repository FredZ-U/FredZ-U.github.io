Go Chop, Unity / C#

Game Engineering/ UI Design
October 2023

Go chop is a 3D Multiplayer Action online game (Only for local network). Players need control character to fight other players in the Colosseum. Players can freely choose weapons for combat. Different weapons have different attack, defense, and counterattack actions. A room in the game can support up to 4 people fighting.

    What I achieved:
  • Designed and developed a 3D multiplayer action online game tailored for local networks where players can compete in a Colosseum setting.
  • Build up network system that able to sync the data, action, and player input at online network.
  • Optimized game to support up to 4 players simultaneously in a single room, ensuring smooth gameplay and interactions.
  • Implemented Network function and combine it into designed UI menu.
